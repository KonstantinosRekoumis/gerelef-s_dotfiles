ffscale-mp4(1) -- scale mp4 video to resolution
===========================================================

## RETURN VALUES
2 on `Not enough, too few or too many arguments`

## SYNTAX
Please note that width x height should not contain spaces.
`ffscale-mp4 <path/to/file> <width x height>`
