ffgiphify(1) -- convert video file to gif from start to duration
===========================================================

## RETURN VALUES
2 on `Not enough or too many arguments`

## SYNTAX
Both start & end must be in Minutes:Seconds format.
`ffgiphify <path/to/file> <start> <end>`
