ffcompress-mp3(1) -- compress mp3 audio 
===========================================================

## RETURN VALUES
2 on `Not enough arguments`

## SYNTAX 
Good bitrate values are, for example, `96k`. 
`ffcompress-mp3 <path/to/file1> <bitrate>`
