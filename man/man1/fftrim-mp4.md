fftrim-mp4(1) -- trim mp4 file from start to end
===========================================================

## RETURN VALUES
2 on `Not enough or too many arguments`

## SYNTAX
Both start & end must be in Minutes:Seconds format.
`fftrim-mp4 <path/to/file> <start> <end>`
